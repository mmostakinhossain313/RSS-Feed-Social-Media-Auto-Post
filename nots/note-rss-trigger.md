# RSS Feed Trigger Note

RSS Trigger = automatically checks website for new content.

Settings I use:
Feed URL = https://feeds.bbci.co.uk/news/technology/rss.xml
Poll Time = Every 1 hour

Any website with RSS feed works.
Most news sites have RSS.
Just add /rss.xml or /feed at the end of URL.

One mistake I made:
I used wrong RSS URL format.
Fix: Always test RSS URL in browser first.
If XML data shows = URL is correct.
